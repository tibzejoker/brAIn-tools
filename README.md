# brAIn-tools

General-purpose capability nodes for [brAIn](https://github.com/tibzejoker/brAIn).

| Node | Purpose |
|---|---|
| `terminal` | Shell command execution. Streams stdout/stderr back to the bus. |
| `calc-py` | Python expression evaluator (web transport, runs in a sidecar). |
| `http-bridge` | HTTP I/O — fetch URLs, post payloads, surface results on the bus. |

Install via the [brAIn marketplace](https://github.com/tibzejoker/brAIn-store).
